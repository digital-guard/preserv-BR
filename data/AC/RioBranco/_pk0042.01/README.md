Abreviaturas raras usadas em eixos:
* RO, RD = Rodovia (equivalente de VIA para os lotes)
* RM = Ramal (equivalente de RM para os lotes)
* ES, EST = Estrada
* BR = Rodovia BR. Se for "BR VIA VERDE", é o equivalente de VIA VERDE dos lotes.
* BC = Beco
* ALA, ALAM = Alameda
